# Prototipo MVP "App Reuniones Online"

### Requerimientos
- Node >= 16.13.1
- npm >= 8.1.2

### Inicio

Instalar Dependencias

```
npm install
```

Correr Enviroment

```
npm run dev
```

La aplicacion se puede visualizar http://localhost:5050
